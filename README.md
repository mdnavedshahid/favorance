# Personal Website with Admin Panel

A complete personal website with image gallery and admin panel for managing images.

## Features

- **Beautiful Gallery**: Responsive grid layout with hover effects
- **Secure Admin Panel**: Login system to manage images
- **Image Management**: Upload and delete images easily
- **Responsive Design**: Works perfectly on all devices
- **Modern UI**: Gradient backgrounds, animations, and icons
- **MySQL Database**: Secure data storage
